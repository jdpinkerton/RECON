# RECON (Reading and Contrast)

This repository provides supplementary digital materials associated with the
RECON (Reading and Contrast) textual comparison workflow used in the doctoral
dissertation:

Jonathan D. Pinkerton, *Marlowe's Playbooks*, Universit√© de Lille / University of Kent

## Status and authority

The authoritative record of the methodology, analytic logic, and results produced
using RECON is the submitted dissertation and its deposited supplementary files.

This repository is provided to support transparency, replication, and future
methodological reuse.

## Contents (high level)

- `RECON v3.1/`  
  Browser-based textual comparison tool as used during the analyses reported
  in the dissertation.

- `Scoped Hero and Leander files/`  
  Scoped comparison input texts used for RECON runs.

- `comparison data/`  
  JSON outputs from RECON comparison runs, including full exports and
  aggregated summaries.

Further implementation details are documented within the software directory.

## Versioning

This repository is cited in the dissertation by a fixed release tag and commit
hash. Subsequent development does not alter the analytic record preserved in
the thesis deposit.
